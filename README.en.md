# CSV2Excel

#### Description
Convert CSV file to Excel

#### Get started

1. [download](https://github.com/ZXS66/csv2excel/releases) （currently only support `win-x64` version, you may clone this repo and build the solution yourself）
2. run below command
    ```
    ./CSV2Excel.exe ./test.csv
    ```
